# voro_modes — the saved faces of the ▦ glass

The owner (2026-07-11): *"it'd be nice to save these modes we're evolving through... this one is
 nice and gem-looking with the w/C%Track in the center of each cell, surrounded by its organs
  like a cell-wall around its nucleus."*

Each pass of the ▦ sub-graph engine replaced the last IN CODE, but every face is anchored by its
 commit and (where the 🩻 instrument existed) a live SVG shot of VoroScape's final beat.  To revive
  a face: `git show <commit>:src/lib/O/Cytui.svelte` (render-only — any of them runs against
   current fixtures).  To shoot a new one: `node scripts/runner_shot.mjs --svg` against a live
    runner with a scape up (op:'svg' ships the glass layer with its CSS baked in — greppable).
 The full design record per pass: `../Voro_vtuffing.md` §🎋 v2, passes first→sixth.
